# pyBinSim_AppExamples
Different applications and using pyBinSim 



## Install pyBinSim

    $ conda create --name binsim35 python=3.5 numpy scipy
    $ source activate binsim35  # only activate binsim35 on win
    $ pip install pybinsim
        
## Start Demo1
    $ cd demo1/
    $ python play_demo.py
