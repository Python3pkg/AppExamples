# pyBinSim_AppExamples
Different applications using pyBinSim 



## Install pyBinSim

    $ conda create --name binsim35 python=3.5 numpy scipy
    $ source activate binsim35  # on windows: activate binsim35
    $ pip install pybinsim
        
## Start Demo1
    $ cd demo1/
    $ python play_demo.py
